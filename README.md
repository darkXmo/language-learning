# Language-Learning 个人语言学习网站

总结了我个人的英语学习习惯，写的一个网站，适用于我学习包括英语在内的多门语言（其实也就是英语和日语）。

## 使用技术

`Vue3` + `Vite` + `Typescript` + `Vant` + `cal-rule`

## 开发环境说明

推荐使用 `pnpm` 作为项目包管理工具

```bash
pnpm i
```

## 运行和部署

运行

```bash
npm run dev
```

部署

```bash
npm run build //正式环境
```
